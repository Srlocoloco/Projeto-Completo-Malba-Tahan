CSS Básico

Introdução ao CSS

CSS (Cascading Style Sheets) é a linguagem de estilo usada para descrever a apresentação de um documento HTML.

  • Cascading: Estilos são aplicados em cascata
  • Style: Define a aparência dos elementos
  • Sheets: Organizado em folhas de estilo

Sintaxe Básica

seletor {
    propriedade: valor;
}

/* Exemplo */
h1 {
    color: blue;
    font-size: 24px;
    margin: 10px;
}

Seletores Básicos

/* Seletor de elemento */
p {
    color: red;
}

/* Seletor de classe */
.destaque {
    background: yellow;
}

/* Seletor de ID */
#titulo {
    font-size: 32px;
}

/* Seletor de atributo */
input[type="text"] {
    border: 1px solid gray;
}

/* Seletor de descendente */
div p {
    margin: 10px;
}

/* Seletor de filho direto */
div > p {
    padding: 5px;
}

/* Seletor de irmão adjacente */
h2 + p {
    font-style: italic;
}

/* Seletor de irmão geral */
h2 ~ p {
    color: blue;
}

Propriedades de Texto

p {
    /* Fonte */
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
    line-height: 1.5;

    /* Texto */
    color: #333;
    text-align: center;
    text-decoration: underline;
    text-transform: uppercase;
    text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
    letter-spacing: 1px;
    word-spacing: 2px;
}

Box Model

div {
    /* Dimensões */
    width: 300px;
    height: 200px;
    
    /* Margens */
    margin: 10px;
    margin-top: 20px;
    margin-right: 15px;
    margin-bottom: 20px;
    margin-left: 15px;
    
    /* Padding */
    padding: 15px;
    padding-top: 20px;
    padding-right: 15px;
    padding-bottom: 20px;
    padding-left: 15px;
    
    /* Borda */
    border: 2px solid black;
    border-radius: 10px;
    
    /* Box Sizing */
    box-sizing: border-box;
}

CSS Intermediário

Layout com Flexbox

.container {
    display: flex;
    flex-direction: row;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
    gap: 20px;
}

.item {
    flex: 1;
    flex-grow: 1;
    flex-shrink: 0;
    flex-basis: 200px;
    align-self: flex-start;
}

Layout com Grid

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;
    gap: 20px;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
}

.grid-item {
    grid-area: main;
    grid-column: span 2;
    grid-row: 2 / 4;
}

Posicionamento

.elemento {
    position: relative;
    top: 10px;
    left: 20px;
    z-index: 1;
}

.elemento-fixo {
    position: fixed;
    bottom: 20px;
    right: 20px;
}

.elemento-absoluto {
    position: absolute;
    top: 0;
    left: 0;
}

.elemento-sticky {
    position: sticky;
    top: 0;
}

Transformações

.elemento {
    /* Translação */
    transform: translate(100px, 50px);
    
    /* Rotação */
    transform: rotate(45deg);
    
    /* Escala */
    transform: scale(1.5);
    
    /* Inclinação */
    transform: skew(20deg);
    
    /* Múltiplas transformações */
    transform: translate(100px, 50px) rotate(45deg) scale(1.5);
}

CSS Avançado

Animações

@keyframes exemplo {
    0% {
        transform: translateX(0);
    }
    50% {
        transform: translateX(100px);
    }
    100% {
        transform: translateX(0);
    }
}

.elemento {
    animation: exemplo 2s infinite;
    animation-timing-function: ease-in-out;
    animation-delay: 1s;
    animation-direction: alternate;
}

Variáveis CSS

:root {
    --cor-primaria: #007bff;
    --cor-secundaria: #6c757d;
    --espacamento: 20px;
    --fonte-principal: 'Arial', sans-serif;
}

.elemento {
    color: var(--cor-primaria);
    margin: var(--espacamento);
    font-family: var(--fonte-principal);
}

Media Queries

/* Dispositivos móveis */
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

/* Tablets */
@media (min-width: 769px) and (max-width: 1024px) {
    .container {
        grid-template-columns: repeat(2, 1fr);
    }
}

/* Orientação do dispositivo */
@media (orientation: landscape) {
    .container {
        height: 100vh;
    }
}

Pseudo-classes e Pseudo-elementos

/* Pseudo-classes */
a:hover {
    color: red;
}

input:focus {
    border-color: blue;
}

li:first-child {
    font-weight: bold;
}

/* Pseudo-elementos */
p::first-line {
    font-variant: small-caps;
}

p::before {
    content: "→ ";
}

p::after {
    content: " ←";
}
